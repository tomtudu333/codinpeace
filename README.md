# CodinPeace 🌿

**Zoom into your code. Find its calm.**

CodinPeace is an infinite canvas for understanding code. From 50,000 feet to the function level — modules become cards, features unfold inside them, and functions wait deeper still. All with a single continuous zoom. Like a meditation on your architecture.

No discrete zoom levels. No jarring transitions. Just you, the scroll wheel, and the quiet satisfaction of seeing how things fit together.

## Why CodinPeace?

Most code tools show you one thing at a time — a file, a tree, a graph. CodinPeace shows you everything at once, then lets you **zoom into what matters**. Like Google Maps for your codebase: see the continent, then the city, then the street, all in one seamless motion.

- **Infinite canvas** — pan, zoom, explore. Nothing is hidden.
- **Continuous semantic zoom** — modules → features → functions, all in one fluid motion. No state machines, no breakpoints.
- **Boxes inside boxes** — feature cards nest inside module cards. Function cards nest inside feature cards. Your architecture, physically.
- **Dark & light themes** — because your eyes deserve a break.
- **Real data, real insight** — metrics, complexity, health scores at every level.

## Quick Start

```bash
npm install
npm run dev
```

Open the browser and start zooming. Space+drag to pan. Scroll to zoom. Click a module to inspect it.

CodinPeace believes that understanding your codebase shouldn't feel like work. It should feel like **exploration**. Zoom out to breathe. Zoom in to focus. The map is the territory.
